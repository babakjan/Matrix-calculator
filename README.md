# Matrix-calculator

![thumbnail](matrixCalculatorScreenshot.png)


### Description:
    -   calculator can add, subtract or multiply two 3*3 matrices.
    -   used technologies: PHP, HTML, CSS
### Usage:
    -   fill matrix A and B, choose operator
    -   press calculate
    -   result matrix is C
### How to run:
    -   run index.php on any php server
